# Солнечная система

Информационная программа о Солнечной системе с движущимися планетами.

## Автор
Руш Егор

## Версия
1.0

## Описание
Программа визуализирует Солнечную систему с движущимися планетами. 
При нажатии на планету отображается информация о её характеристиках.

## Функциональность
- Визуализация Солнечной системы с 8 планетами
- Движение планет по орбитам с разной скоростью
- Отображение информации при нажатии на планету:
  - Орбитальные характеристики (расстояние, период)
  - Физические характеристики (радиус)
  - Температура
  - Состав атмосферы
- Регулировка скорости движения всех планет
- Легенда с цветами планет

## Управление
- Нажмите на планету - отобразится информация
- Нажмите на Солнце - информация о Солнце
- Ползунок скорости - изменение скорости
- Кнопка "Сбросить скорость" - возврат к 1.0

## Требования
- Python 3.6 или выше
- Tkinter (встроенная библиотека)
